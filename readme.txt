Accept payments from all major polish banks directly on your WooCommerce using Przelewy24. Gateway supports PLN,CZK, GBP, EUR currencies.

== Description ==

Brama płatności dla pluginu Woocommerce z wykorzystaniem serwisu Przelewy24.pl

== Installation ==
Aby wtyczka działała prawidłowo wymagane są pola:
1) ID sprzedawcy – pole obowiązkowe, Twój ID otrzymany podczas zakładania konta  Przelewy24.pl
2) Klucz CRC  – należy wpisać kod ustawiony w Panelu Przelewy24.pl.
